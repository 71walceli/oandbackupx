Usage notes

Backup modes: An app can be backed up with only its apk, only its data or both. An app not currently installed can only be restored if the apk is present.

Διαγραφή αντιγράφου: Διαγράφει τα αρχεία των αντιγράφων για την επιλεγμένη εφαρμογή.

Απεγκατάσταση: Κάπως πιο δυναμική από μία κανονική απεγκατάσταση. Επιπλέον από την εκτέλεση μίας κανονικής απεγκατάστασης μέσω εντολών του android, η απεγκατάσταση από το OAndBackupX διαγράφει αρχεία που η εφαρμογή ίσως έχει αφήσει πίσω στο /data/app-lib/.

Ενεργοποίηση /Απενεργοποίηση: Χρησιμοποιεί το script του android `pm` για να ενεργοποιήσει ή απενεργοποιήσει μία εφαρμογή. Απενεργοποιώντας μία εφαρμογή αφαιρείται από το κανονικό περιβάλλον χρήστη χωρίς να απεγκαθίσταται. Αυτό μπορεί να χρησιμοποιηθεί για την ενεργοποίηση ή απενεργοποίηση μίας εφαρμογής για πολλαπλούς χρήστες καθε φορά (εάν η συσκευή έχει πολλαπλούς χρήστες ενεργούς). Οι χρήστες αναγνωρίζονται με μία ταυτότητα. 0 είναι ο πρώτος χρήστης (κάτοχος).

Πολλαπλοί χρήστες: Οι πολλαπλοί-χρήστες είναι ακόμα κάπως πειραματικό αλλά πρέπει να λειτουργεί. Όταν γίνεται επαναφορά σε ένα σενάριο πολλαπλών-χρηστών, το `pm install -r $apk` καλείται και ακολούθως η εφαρμογή απενεργοποιείται για κάθε χρήστη που διαθέτει την εφαρμογή στο /data/system/user/$user/package-restrictions.xml (εκτός εάν η εφαρμογή είναι καταγεγραμμένη ως ενεργοποιημένη="1").
Αυτό μπορεί να δημιουργήσει προβλήματα για χρήστες που θα εγκαταστήσουν την ίδια εφαρμογή σε κάποια μεταγενέστερη φάση, όμως είναι απαραίτητο να αποτραπεί η εγκατάσταση της εφαρμογής για όλους τους χρήστες ταυτόχρονα. Το μενού περιβάλλοντος διαθέτει μία επιλογή για ενεργοποίηση ή απενεργοποίηση εφαρμογών που μπορεί να χρησιμοποιηθεί εάν οι άλλοι χρήστες καταστούν ανίκανοι να χρησιμοποιήσουν μία συγκεκριμένη εφαρμογή εξαιτίας απενεργοποίησής της κατά την επαναφορά.
Η ενεργοποίηση κι η απενεργοποίηση λειτουργούν μόνον έπειτα από μία αρχική εγκατάσταση (όχι απαραιτήτως από το OAndBackupx) ή επαναφορά της εφαρμογής.
