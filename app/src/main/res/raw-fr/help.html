Notes d'utilisation

Modes de sauvegarde : Il est possible de ne sauvegarder que l'APK d'une app, que ses données, ou bien les deux. Une app qui n'est pas installée ne peut seulement être restaurée si l'APK a été sauvegardée. Trois types spéciaux de données sont pris en charge en plus des données principales : les données protégées par l'appareil, les données externes et les données obb.

Effacer la sauvegarde : Supprime les fichiers de sauvegarde pour l'app choisie.

Désinstaller : Un peu plus aggressif qu'une désinstallation normale. En plus d'une désinstallation normale via les commandes Android, OAndBackupX supprime les fichiers que l'app aurait pu laisser dans /data/app-lib/.

iActiver / Désactiver : Utilise le script Android `pm` pour activer ou désactiver une app. Désactiver une app la retire de l'interface utilisateur normale sans la désinstaller. Il est aussi possible d'activer ou désactiver une app pour plusieurs utilisateurices à la fois (si l'appareil est configuré pour). Les utilisateurices sont identifié⋅es par un identifiant. Le premier est le numéro 0 (propriétaire).

Désinstaller : Un peu plus aggressif qu'une désinstallation normale. En plus d'une désinstallation normale via les commandes Android, OAndBackupX supprime les fichiers que l'app aurait pu laisser dans /data/app-lib/.

System-less apps: backup and restore of system-less apps installed as Magisk module is not supported and trying it could lead to failure in the system and forcing you to format and flash anew.

Navigation : La barre de navigation a « Paramètres », « Accueil », « Sauvegarde par lots », « Restauration de lots » et « Sauvegardes d’horaire » (dans cet ordre). Le nombre d’applications mises à niveau après leur dernière sauvegarde sera présenté sous forme de notification sur le bouton « Accueil ».

Apps using Keystore: Android has an encrypted storage for sensitive data. Apps that make use of this to store login data or some keys for its function can't be restored successfully. Two examples of such apps are Signal and Element(Riot).
