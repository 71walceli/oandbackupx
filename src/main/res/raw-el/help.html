σημειώσεις χρήσης

λειτουργίες αντιγράφου: μπορεί να δημιουργηθεί αντίγραφο για μία εφαρμογή με μόνον το apk, μόνον τα δεδομένα ή και τα δύο. μία εφαρμογή που δεν είναι προς το παρόν εγκατεστημένη μπορεί να επαναφερθεί μόνον εάν το apk είναι παρόν.

μενού περιβάλλοντος: πατήστε παρατεταμένα σε ένα αντικείμενο στη λίστα εφαρμογών για να μεταβείτε στο μενού περιβάλλοντος.

διαγραφή αντιγράφου: διαγράφει τα αρχεία αντιγράφου για την επιλεγμένη εφαρμογή.

απεγκατάσταση: κάπως πιο δυναμική από μια κανονική απεγκατάσταση. περαιτέρω από μια κανονική απεγκατάσταση μέσω εντολών android, η απεγκατάσταση από το OAndBackupX διαγράφει αρχεία που μπορεί η εφαρμογή να έχει αφήσει πίσω στο /data/app-lib/. λειτουργεί επίσης σε εφαρμογές συστήματος.

ενεργοποίηση / απενεργοποίηση: χρησιμοποιεί το σενάριο του android `pm` για να ενεργοποιήσει ή απενεργοποιήσει μία εφαρμογή. απενεργοποιώντας μια εφαρμογή αφαιρείται από τον κανονικό χρήστη χωρίς να απεγκαθίσταται. αυτό μπορεί να χρησιμοποιηθεί για την ενεργοποίηση ή απενεργοποίηση μίας εφαρμογής για πολλαπλούς χρήστες καθε φορά (εάν η εφαρμογή έχει πολλούς χρήστες ενεργούς). οι χρήστες αναγνωρίζονται με μία ταυτότητα. 0 είναι ο πρώτος χρήστης (κάτοχος).

πολλαπλοί χρήστες: οι πολλαπλοί-χρήστες είναι ακόμα κάπως πειραματικό αλλά πρέπει να λειτουργεί. όταν γίνεται επαναφορά σε ένα σενάριο πολλαπλών-χρηστών, `pm install -r $apk` καλείται και ακολούθως η εφαρμογή απενεργοποιείται για κάθε χρήστη που διαθέτει την εφαρμογή στο /data/system/user/$user/package-restrictions.xml (εκτός εάν η εφαρμογή είναι καταγεγραμμένη ως ενεργοποιημένη="1").
αυτό μπορεί να δημιουργήσει προβλήματα για χρήστες που θα εγκαταστήσουν την ίδια εφαρμογή σε κάποια μεταγενέστερη φάση, όμως είναι απαραίτητο να αποτραπεί η εγκατάσταση της εφαρμογής για όλους τους χρήστες ταυτόχρονα. το μενού περιβάλλοντος διαθέτει μία επιλογή για ενεργοποίηση ή απενεργοποίηση εφαρμογών που μπορεί να χρησιμοποιηθεί εάν οι άλλοι χρήστες καταστούν ανίκανοι να χρησιμοποιήσουν μία συγκεκριμένη εφαρμογή εξαιτίας απενεργοποίησης της κατά την επαναφορά.
η ενεργοποίηση και απενεργοποίηση λειτουργούν μόνο μετά από μία αρχική εγκατάσταση (όχι απαραιτήτως από το OAndBackupx) ή επαναφορά της εφαρμογής.
